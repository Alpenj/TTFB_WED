✨

신규 기능 (New Features)

  • ✓
    경기 명단 (Linup) 확인

    일정 페이지의 각 경기 카드에 명단 버튼이 추가되었습니다. 클릭 시 해당 경기의 선발/교체 명단과 개인 기록(득점/도움/경고)을 한눈에 볼 수 있습니다.

  • ✓
    인앱 가이드 (In-App Guide)

    하단 네비게이션바의 물음표(?) 버튼을 눌러보세요. 화면 설명과 데이터 관리 방법을 앱 내에서 바로 확인할 수 있습니다.

  • ✓
    연습경기 분리

    '전체' 탭에서 연습경기를 제외하고, 상단의 [연습경기] 전용 탭에서만 모아볼 수 있도록 정리했습니다.

v1.2 추가 사항

  • • 무실점(Clean Sheets) 컬럼 추가: 기록실 테이블에서 무실점 기록을 확인하고 정렬할 수 있습니다.
🛠️

개선 사항 (Improvements)

  • 모바일 스크롤 고정: 스마트폰에서 상하단 바가 덜컹거리지 않도록 화면을 고정했습니다.
  • 데이터 정확도 향상: 선발/교체 명단이 실제 데이터와 일치하지 않던 문제를 수정했습니다.
  • UI 디자인 통일: 기록실(Stats) 페이지의 각 섹션 제목 크기와 여백을 보기 좋게 통일했습니다.
  • 경기 당일 표시: 경기 당일에는 D-Day 대신 "경기 당일" 라벨이 표시됩니다.
  • 모바일 레이아웃: 홈 화면 요약 카드를 모바일에서 보기 편한 2열 구조로 변경했습니다.
📝

데이터 관리 가이드 (매니저)

앱 내 가이드(?) 버튼의 [데이터 관리] 탭에서도 동일한 내용을 확인하실 수 있습니다.

1. 경기 영상 버튼 (I열)

  • 파란 버튼: I열에 유튜브 링크 입력.
  • 빨간 버튼: I열을 비워두면 자동 적용.

2. 득점-도움 연결 (비고란)

  • 득점자와 도움 선수의 [기록원장] 시트 [비고] 란에 같은 태그(예: G1)를 입력하면 자동으로 연결됩니다.
