홈 (Home)

팀의 현재 상황을 한눈에 파악하세요.

  • 요약 카드 개편: 경기 수(Matches) -> 전적(W-D-L) -> 승률 순으로 배치 변경.
  • 다음 경기 & 최근 결과: '연습경기'는 제외하고 리그/컵 공식 경기 위주로 표시 (전체 보기 시).
  • 경기장 지도: 🏟️ 아이콘 클릭 시 네이버/카카오/TMAP 연동.
  • 전적 분석: 상대 팀별 / 구장별 승률 데이터 (접기/펼치기 가능).
📅

경기 일정 (Schedule)

직관적인 리스트로 지난 경기와 예정된 일정을 확인하세요.

  • 필터링 개선: '전체' 탭에서는 연습경기를 제외하고 공식 일정만 표시합니다.
  • 스코어 표기: 점수 옆에 (승), (무), (패) 라벨 추가.
  • 상태 색상: 예정된 경기는 파란색 라인으로 구분.
  • 상세 결과: 승부차기(PK) 결과와 득점/도움 선수 실명 표기.
  • 경기 영상: 영상 버튼 추가 (파란색: 시청 가능, 빨간색: 미등록).
📊

기록실 (Records)

데이터 분석의 핵심 페이지입니다.

  • 부문별 랭킹: 득점, 도움, 출전, 경고, 퇴장, 자책골(Own Goal) 순위.
  • 개인 히스토리 팝업: 선수 이름을 클릭하면 언제, 누구를 상대로 기록했는지 팝업으로 상세히 보여줍니다.
  • 종합 스탯 테이블:
    • - 모든 항목 정렬(Sorting) 지원.
    • - Update! 출전 기록 세분화: 선발(Starts) / 교체(Subs) 분리 표기.
📝

데이터 관리 가이드 (매니저 필독)

구글 스프레드시트의 [경기 일정] 시트를 수정하면 대시보드에 즉시 반영됩니다.

1. 경기 결과 및 상태 설정 (H열)

  • 경기 완료: H열에 스코어를 2:1 형식으로 입력하세요. (승/패 구분 자동)
  • 진행 예정: H열을 비워두면 자동으로 '진행 예정' 상태가 되며, 파란색 테두리로 표시됩니다.

2. 경기 영상 버튼 설정 (I열)

  • 영상 없음 (빨간 버튼): I열을 비워두면 됩니다.
  • 영상 있음 (파란 버튼): I열에 유튜브 등 영상 URL을 입력하면 자동으로 파란색 버튼으로 변경됩니다.
